Conclusion

You've completed the CyberEd module on Steganography! Let's review what we covered

  • Steganography is the act of hiding something inside something else
  • Often, steganography is used to hide a message in a public statement or image
  • JPEG images (among others) allow for extra information in the file which doesn't affect the picture. This is a good place to hide a message
  • Images are made of pixels, which are represented by numbers
  • We can encode data in images by modifying pixels, and decode the message by finding the difference from the original
  • When hiding a message by modifying pixels, you have to make sure you don't modify them too much, or someone might notice

We started by showing a simple example of how you could hide a message inside a paragraph of text, but it was pretty easy to find messages hidden that way, so we looked at hiding messages in pictures.

When pictures allow adding arbitrary extra information, you can hide whatever message you want, and as you saw, it doesn't affect the picture at all. The only way to find the hidden message is to look at the raw data from the picture, or use a tool to find it for you.

However, some formats don't allow comments, so we showed how you can modify the picture itself to hide the data. This has the advantage of not requiring a special image format, but the disadvantage of needing to have the original image before you can decode the message. In addition, you saw how changing the value of a pixel can make it noticably different, which is something to remember when hiding a message this way.

There are two big topics that we mentioned but did not discuss in this module, and those are color channels and bit planes. When pictures are in color, they actually contain 3x as much information, and when that's the case it's much easier to hide data in the picture, because you have to make bigger changes to the picture before it's noticable. If you have further interest in image steganography, we encourage you to go find more information about it online.

If you want another chance to encode more text or images, continue to the next page!
