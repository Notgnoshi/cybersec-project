Introduction

Hashing is the process of taking some data and creating a condensed representation of that data which is semi-unique to that specific data. This condensed representation is called the hash of the data, and in cyber-security contexts, it is often called the message digest. This module will explore some of the commonly-used hashing methods, and demonstrate the usages of message digests for security and data integrity.

Module Goals

  • Motivate the usage of message digests
  • Describe what a hash function is
  • Show examples of well-known hash functions
  • Demonstrate usage of message digests
