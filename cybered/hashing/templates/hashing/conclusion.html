Conclusion

You've completed the CyberEd module on hashing! We started by motivating the need for message digests with a story about Alice, Bob, and Chuck, and then learned about what a message digest is, how a hash function makes one, and what makes a hash function good.

After going over the basics, you used a tool to generate message digests for any messages you wanted. If you tested a bunch of different messages, you saw how each hash algorithm generated a digest that was always the same size, and you also saw that even if your message was almost identical to a different message, the digest was completely different.

Finally, you saw how keyed hashing could be used to add a password to your message digest, making it incredibly difficult for someone to fake, even if they had the message you sent and the digest you sent with it.
